# Noloong App GUI Reference

These images are visual references for the planned GPUI-based `noloong app` experience.

- `noloong-logo-reference.png`: app logo direction.
- `profile-settings-jsonc-closed.png`: profile settings with JSONC preview hidden by default.
- `profile-settings-jsonc-open.png`: profile settings with JSONC preview opened as a floating overlay.
- `chat-minimal.png`: minimal chat workspace with right-side floating toolbar.
- `chat-tools-popover.png`: chat workspace with tools popover opened from the floating toolbar.
