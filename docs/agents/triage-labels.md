# Triage Labels

工程技能使用五个标准 triage 角色。本文件定义这些角色在本仓库 issue tracker 中对应的状态字符串。

| 技能角色 | 本仓库状态 | 含义 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 需要维护者评估 |
| `needs-info` | `needs-info` | 等待报告者补充信息 |
| `ready-for-agent` | `ready-for-agent` | 描述完整，可以交给 agent 独立处理 |
| `ready-for-human` | `ready-for-human` | 需要人类实现或判断 |
| `wontfix` | `wontfix` | 不会处理 |

当技能提到某个 triage 角色时，使用上表右侧的状态字符串。

如果以后本仓库改用其它状态词汇，只需要修改“本仓库状态”这一列。
